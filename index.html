Admin Guide — how to run the app

Everything an admin needs, panel by panel. Tap a section to expand it.

👥 Members — approve, verify, renew
  1. New enrollments arrive as Pending. Confirm their payment reference against your bank/MoMo statement, then press ✓ Approve. The member's benefits unlock instantly.
  2. ✉ Verify appears for members who couldn't receive their email code — only use it after confirming their identity.
  3. Renewals: members whose year has ended submit a renewal (amount + reference). Pending renewals appear on the Overview page — press ✓ Approve to extend their membership year.
🖼 Images & videos — the two systems
  1. Admin uploads (no GitHub needed): Conservation Gallery photos/videos, payment logos, “How to Pay” welcome media. Use the upload buttons in each panel — changes appear on the site instantly. If an upload fails you will see the exact error message.
  2. Repo files (GitHub): hero slides, brand logos. Upload via Add file → Upload files (drag & drop) on the main branch. Keep both slide-N.jpg (desktop) and slideNsm.jpg (phone) versions.
💳 Payment details & impact list

Everything the public “How to Pay” section shows is edited here: bank/MoMo numbers, WhatsApp line, channel logos, the welcome video/image, and the “Where your money goes” list (one line per impact: emoji | amount | what it achieves). Saving updates the live site immediately.

🦍 Conservation Gallery

Each card = one species or place (name, IUCN status, story, region, photo or short video, optional Green Card tier link — linked items show as “You help protect …” on that tier). Untick Show on the public page to hide a card without deleting it.

📣 Ad Campaigns

Create a card with title, message, image, link and button label, then set start and end dates — it runs only inside that window and appears as a “Sponsored · UBF” card in the members' feed. The list shows each campaign's status (Scheduled / ● Live / Ended) plus 👁 views and 🖱 clicks.

📅 Events & 🎯 Fundraisers

Events: publish with date, time and location; members RSVP and you see live counts. Fundraisers: set a goal; donations arrive as pending — press ✓ Confirm once the payment is verified and the progress bar grows automatically.

📚 Content, 📢 Announcements, 📄 Reports

Upload Learning Exchange resources, post announcements (optionally emailed), publish financial reports. Items older than 21 days auto-archive out of the main feeds — nothing is deleted; use View archived or the Archive/Unarchive buttons.

🧾 Audit Log & 🔑 Password

The Audit Log records who approved, published, confirmed or deleted what, with timestamps. Every admin should set a personal password under My Password on first login.

🆘 Troubleshooting
  1. Changes not visible? Hard-refresh (Ctrl+Shift+R / pull-down refresh on mobile). Code updates load fresh automatically; images may need one extra refresh.
  2. Upload failed? The red toast shows the exact reason. Files over ~50 MB are rejected — compress videos first.
  3. A member can't sign in? Check their status (pending → approve; inactive → reactivate) and whether their email is verified.
